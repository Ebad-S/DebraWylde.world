## 1. Purpose of Agreement

This agreement defines the scope, deliverables, and payment terms for the design, development, and deployment of the website **DebraWylde.world** for Debra Wylde’s advisory service.

By signing this agreement and submitting the initial deposit, both parties agree to the terms outlined below.

---

## 2. Scope of Work

Serenity Webcrafts will design and develop a professional website for Debra Wylde’s 12-week Transformation Program advisory service. The project includes the following components:

### Website Design & Development

- Responsive website design
- Static HTML architecture with lightweight CSS
- Minimal JavaScript usage for performance and reliability
- Alternating layout design (photo / text sections)

### Core Website Pages

- Homepage
- Program / Services page
- About page
- Contact / Discovery Call page / Calendar booking
- Blog page (for SEO and content publishing)

### Lead Capture System

- Contact / discovery call request form
- Email notifications sent to `hello@DebraWylde.world`
- Confirmation email sent to the user
- Lead data stored in a simple backend database

### Technical Setup

- Domain configuration and DNS setup
- Hosting deployment on Serenity Webcrafts VPS infrastructure
- Basic on-page SEO structure
- Integration with Instagram and other provided social links

### Optional Integrations (based on client preference)

- Discovery call booking integration (e.g., Calendly or Google Calendar)
- Payment integration (e.g., Stripe)
